Software supporting COMBINE archives

Reproducibility in Action!

COMBINE archive

The COMBINE archive library was developed by Frank Bergmann. Excellent tool to add support for COMBINE archives in C# projects.

PySCeS

PySCeS provides a variety of tools for the analysis of cellular systems. It supports COMBINE archives (reading/exporting?)

CombineArchiveWeb

The CombineArchive WebInterface is a web based user interface to create, modify, explore, and share COMBINE archives.

PySCeS CBMPy

PySCeS CBMPy is a new platform for constraint based modelling and analysis. It supports COMBINE archives (reading/exporting?)

SED-ML Web Tools

The SED-ML Web Tool can read and export COMBINE archives. A sophisticated API allows for automatic runs of simulation studies encoded in the archives.

tellurium

Tellurium (te.) is a Python environment based on spyder2 IDE. It is able to consume COMBINE archives.
